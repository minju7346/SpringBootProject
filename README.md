# Spring Boot를 이용한 To Do List 프로그램
> 해당 프로젝트는 개인프로젝트로(2022.01.02~2022.02.15) 진행하였으며, CRUD기능을 구현하여 만든 task관리 프로그램입니다.

## 사용 스택
  
- **Frontend** : Mustache, BootStrap, JQuery
- **Backend** : Spring Boot
- **DB** : Spring Data JPA
 
  
  
  
  <br/><br/><br/><br/>
  
  
  
  
  
## 구현 기능 ( CRUD + 구글 로그인 )

- **To Do task생성**
- **To Do list 출력**
- **To Do task 수정**
- **To Do task 삭제**
- **구글 로그인**
